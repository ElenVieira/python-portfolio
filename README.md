# python-portfolio
Python portfolio with projects involving data analysis, automation, APIs and algorithms.

Este repositório contém projetos desenvolvidos para demonstrar minhas habilidades em Python.

## Tecnologias

- Python
- Pandas
- Matplotlib
- Requests
- JSON

## Projetos

### Sales Data Analysis
Análise de vendas com visualização de dados.

### Web Scraping
Extração de dados de sites usando BeautifulSoup.

### API Weather
Consumo de API de clima utilizando Requests.

## Objetivo

Praticar desenvolvimento em Python, análise de dados e automação.
